- Core Classroom Workflow:
  - Class lifecycle, students, marks, attendance, seating, notes, loaned items, learning skills.

  - Class/student/combined analytics tabs + report outputs.
- Comments and Communication:
  - Comment banks, construction, transfer modes, report-card integration.
- Planning and Publishing:
  - Unit/lesson planner and course description generator.
- Integrations and Exchange:
  - SIS/import/export adapters, external platform transfers.
- Setup/Admin:
  - Calculation options, attendance rules, styles/printer/email/password.
